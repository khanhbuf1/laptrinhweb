CSS Syntax

Quy tắc CSS bao gồm một bộ chọn và một khối khai báo.

Cú pháp CSS

Bộ chọn trỏ đến phần tử HTML bạn muốn tạo kiểu.

Khối khai báo chứa một hoặc nhiều tờ khai được phân tách bằng dấu chấm phẩy.

Mỗi khai báo bao gồm một tên thuộc tính CSS và một giá trị, được phân tách bằng dấu hai chấm.

Nhiều khai báo CSS được phân tách bằng dấu chấm phẩy và khai báo Các khối được bao quanh bởi niềng răng xoăn.

Ví dụ

Trong ví dụ này, tất cả các phần tử 'p' sẽ được căn giữa, với màu đỏ Màu chữ:

p {

color: red;

text-align: center;

}

Giải thích ví dụ

  • P là một bộ chọn trong CSS (nó trỏ đến phần tử HTML bạn muốn tạo kiểu: "tr").
  • color là một tài sản, và là giá trị tài sản red
  • text-alignlà một tài sản, và là giá trị tài sản center
